# 사용자

### 사용자 관련 중요 파일

`/etc/passwd`, `/etc/group`, `/etc/shadow`

### 사용자 관련 주의 사항

사용자 관련 디렉터리는 일반 디렉터리가 아니므로 절대 rm으로 삭제해서는 안됨
사용자 관련 파일은 절대 편집해서는 안됨
사용자 정보와 디렉터리 정보는 반드시 일치가 되어야 함

### 기타

`/etc/skel/`
사용자 생성 시 사용자 계정과 동일한 디렉터리 안에 자동으로 생성되는 파일 또는 디렉터리



# 사용자 관련 명령어

### 기본

**생성** - `useradd [옵션1][값1]...사용자명`

```linux
useradd -c testuser -s /bin/sh user2
```

**수정** - `usermod [옵션1][값1]...사용자명`
**삭제** - `userdel [옵션1][값1]...사용자명`

**비밀번호 변경** - `passwd 사용자명`

### 그룹

**그룹 확인** - `groups 사용자명`
**그룹 만들기** - `groupadd 그룹명`
**그룹 삭제** - `groupdel 그룹명`

+ **그룹 지정**
  `Usermod -G g1 user1` -> -G는 추가그룹 (추가그룹은 다시 한번 똑같이 입력을 하면 추가가 취소됨)
  `Usermod -g g1 user1` -> -g는 기본그룹
